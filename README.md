# Python-Chess-Engine
# ♟️ Python Chess Engine (Web-Based)

A smart chess engine built with Python, integrated with a web interface for interactive gameplay.

## 🚀 Overview
This project combines a custom Python-based chess logic engine with a responsive frontend. It uses a server to process moves and a JavaScript-based UI to render the board, allowing users to play against the AI in real-time.

## 🛠️ Key Features
- **Custom Move Validation:** Server-side logic to enforce standard chess rules.
- **Interactive GUI:** Visual chessboard using `chessboard.js` and custom piece assets.
- **Move Analysis:** `analyzer.py` evaluates board positions to find the best moves.
- **Opening Book:** Includes `openings.py` for standard opening strategies.

## 💻 Tech Stack
- **Backend:** Python (Server & Logic)
- **Frontend:** HTML, CSS, JavaScript (Chessboard.js, jQuery)
